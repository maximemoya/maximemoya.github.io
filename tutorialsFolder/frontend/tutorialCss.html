let's use CSS to make style :
            
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<body style="text-align: center;">
    your body tags and text here
</body>

</html>

At line 7, we added in body tag the parameter: style="text-align: center;"
  -> it will make text align center inside the body tag

We will create a div tag in our body tag : 
            
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<body style="text-align: center;">
    your body tags and text here
    <div style="color:green; text-align: right;">
        your div tags and text here
    </div>
</body>

</html>
Good, now we will move styles out of the tags : 
            
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<style>
    body {
        text-align: center;
    }

    div {
        color: green;
        text-align: right;
    }
</style>

<body>
    your body tags and text here
    <div>
        your div tags and text here
    </div>
</body>

</html>
Excellent it's better now !
  -> we will create a specific style refering parameter class="" : 
            
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<style>
    body {
        text-align: center;
    }

    div {
        color: green;
        text-align: right;
    }

    .MyStyle {
        color: blueviolet;
        text-align: left;
        font-size: xx-large;
    }
</style>

<body class="MyStyle">
    your body tags and text here
    <div>
        your div tags and text here
    </div>
</body>

</html>
Good we can now put this style in an other file !
  -> create a file 'myStyle.css' in the same folder as 'index.html'.
  -> go inside and write your styles (body / div / .MyStyle) : 
            
--------------------------------
--- inside our: myStyle.css ----
--------------------------------

body {
    text-align: center;
}

div {
    color: green;
    text-align: right;
}

.MyStyle {
    color: blueviolet;
    text-align: left;
    font-size: xx-large;
}
Very Good ! We can import our CSS file now :
  -> add : <link rel="stylesheet" href="myStyle.css">' in header
  -> remove <style>...</style> (red part) 
            
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link rel="stylesheet" href="myStyle.css">
</head>

<style>
    body {
        text-align: center;
    }

    div {
        color: green;
        text-align: right;
    }

    .MyStyle {
        color: blueviolet;
        text-align: left;
        font-size: xx-large;
    }
</style>

<body class="MyStyle">
    your body tags and text here
    <div>
        your div tags and text here
    </div>
</body>

</html>
Excellent, finally it will look like that :
