HTML Tutorial


<by writing 'doc' in vscode you can write your first basic HTML code> 
            
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<body>
    your body tags and text here
</body>

</html>

We can change the title of the page as 'ourFirstHtml' : 
            
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ourFirstHtml</title>
</head>

<body>
    your body tags and text here
</body>

</html>

We can add a title with tag h1 : 
            
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ourFirstHtml</title>
</head>

<body>    
    <h1>Main Title</h1>

    your body tags and text here
</body>

</html>

We can add a sub-title with tag h2 : 
            
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ourFirstHtml</title>
</head>

<body>
    <h1>Main Title</h1>
    
    <h2>Sub-title</h2>

    your body tags and text here
</body>

</html>

We can add an unordered list (.-.-.) and ordered list (1-2-3) with items inside : 
            
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ourFirstHtml</title>
</head>

<body>
    <h1>Main Title</h1>

    <h2>Sub-title</h2>
    
    <ul>
        <li>item_1</li>
        <li>item_2</li>
        <li>item_3</li>
    </ul>
    
    <ol>
        <li>item_1</li>
        <li>item_2</li>
        <li>item_3</li>
    </ol>

    your body tags and text here
</body>

</html>

We can add a division to group tags : 
             
 <html lang="en">
 
 <head>
     <meta charset="UTF-8">
     <meta name="viewport" content="width=device-width, initial-scale=1.0">
     <title>ourFirstHtml</title>
 </head>
 
 <body>
    <h1>Main Title</h1>
 
    <h2>Sub-title</h2>
     
    <div>
    
        <ul>
            <li>item_1</li>
            <li>item_2</li>
            <li>item_3</li>
        </ul>
        
        <ol>
            <li>item_1</li>
            <li>item_2</li>
            <li>item_3</li>
        </ol>

    </div>
 
    your body tags and text here
 </body>
 
 </html>

We can add an anchor to go to an other html and add br to enter a line :

</div>

</body>

</html>
