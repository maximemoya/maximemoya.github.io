HTML Tutorial


<by writing 'doc' in vscode you can write your first basic HTML code> 
            
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<body>
    your body tags and text here
</body>

</html>

Congratulations, this is enough to start your first website !

  -> Save it in a 'index.html' file.
  -> Use directly in a browser by drag and drop.
  -> Upload it in your server.
